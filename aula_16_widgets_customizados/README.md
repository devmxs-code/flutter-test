# Aula 16 - Widgets Customizados

Projeto de aprendizado demonstrando criação de widgets customizados e reutilizáveis no Flutter.

## Conceitos Demonstrados

- Criação de widgets customizados
- Reutilização de código
- Parâmetros e propriedades de widgets
- Composição de widgets
- Boas práticas de organização

## Funcionalidades

- Widget CardCustomizado reutilizável
- Múltiplas instâncias do mesmo widget
- Interatividade com callbacks
- Design consistente

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Widget customizado CardCustomizado

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como criar widgets customizados
- Como passar parâmetros para widgets
- Como reutilizar widgets
- Como organizar código
- Boas práticas de composição de widgets
