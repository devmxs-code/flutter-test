# Aula 15 - Animações

Projeto de aprendizado demonstrando criação de animações no Flutter.

## Conceitos Demonstrados

- AnimationController
- Tween e CurvedAnimation
- AnimatedBuilder
- Animações de tamanho e posição
- Curves para efeitos suaves

## Funcionalidades

- Animação de círculo que cresce e diminui
- Controle de play/pause da animação
- Animação suave com Curves

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Exemplo de animação com AnimationController

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como criar animações no Flutter
- Como usar AnimationController
- Como usar Tween para valores animados
- Como usar AnimatedBuilder
- Como aplicar Curves para animações suaves
