# App Viagens

Aplicação Flutter de viagens mostrando belas cidades e destinos brasileiros.

## Funcionalidades

- Explorar cidades brasileiras
- Mostras belas de cidades
- Navegação entre diferentes destinos
- Múltiplas rotas e telas
- Informações e imagens de cidades

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação com roteamento
- `lib/home.dart` - Tela inicial
- `lib/cidades.dart` - Tela de cidades
- `lib/models/` - Modelos de dados de cidades (São Paulo, Rio de Janeiro, Maranhão, Fortaleza, Bahia)

## Rotas

- `/` - Tela inicial
- `/segunda` - Primeiro destino
- `/terceira` - Segundo destino
- `/quarta` - Terceiro destino
- `/quinta` - Quarto destino
- `/sexta` - Quinto destino

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Rotas nomeadas para navegação
