# App ListView

Projeto Flutter demonstrando o uso do widget ListView e listas roláveis.

## Funcionalidades

- Exemplos de widget ListView
- Implementação de lista rolável
- Customização de itens de lista

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Aplicação com exemplos de ListView

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
