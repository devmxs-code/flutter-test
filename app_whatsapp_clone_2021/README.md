# Clone WhatsApp 2021

Aplicação clone moderna do WhatsApp construída com Flutter, apresentando uma interface completa de mensagens.

## Funcionalidades

- Interface de chat com bolhas de mensagem
- Tela de status
- Tela de chamadas
- Tela de configurações
- Integração de câmera
- Navegação inferior
- AppBar customizado
- Design de UI moderno correspondente ao WhatsApp

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/screens/` - Telas da aplicação (chats, status, chamadas, configurações, câmera)
- `lib/components/` - Componentes de UI reutilizáveis
- `lib/models/` - Modelos de dados

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Ícones customizados
