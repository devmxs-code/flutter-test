# Aula 9 e 10 - Múltiplas Rotas

Projeto de aprendizado Flutter demonstrando múltiplas rotas e padrões de navegação.

## Funcionalidades

- Configuração de múltiplas rotas
- Rotas nomeadas
- Navegação entre múltiplas telas
- Parâmetros de rota

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação com múltiplas rotas
- `lib/Primeira.dart` - Primeira tela
- `lib/Segunda.dart` - Segunda tela
- `lib/Terceira.dart` - Terceira tela
- `lib/Quarta.dart` - Quarta tela
- `lib/Quinta.dart` - Quinta tela

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Rotas nomeadas
