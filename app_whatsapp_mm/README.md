# WhatsApp MM

Aplicação de mensagens estilo WhatsApp com integração de câmera e UI moderna.

## Funcionalidades

- Interface de chat
- Página de status
- Página de chamadas
- Página de câmera para tirar fotos
- Compositor de mensagens
- Gerenciamento de contatos
- Design moderno estilo WhatsApp

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/screens/` - Telas da aplicação (home, chats, status, chamadas, câmera)
- `lib/widgets/` - Widgets reutilizáveis
- `lib/data/` - Modelos de dados (contatos, chamadas, status)

## Tecnologias Utilizadas

- Flutter SDK
- Pacote Camera
- HexColor para gerenciamento de cores
- Material Design
