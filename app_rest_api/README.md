# App REST API

Aplicação Flutter demonstrando integração com API REST e busca de dados.

## Funcionalidades

- Integração com API REST
- Busca e exibição de dados
- Tratamento de requisições HTTP
- Interface moderna para exibir dados da API

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/screen/home.dart` - Tela principal exibindo dados da API

## Tecnologias Utilizadas

- Flutter SDK
- Pacote HTTP (para chamadas de API)
