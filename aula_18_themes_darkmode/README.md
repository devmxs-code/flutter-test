# Aula 18 - Themes e Dark Mode

Projeto de aprendizado demonstrando criação de temas e implementação de modo escuro no Flutter.

## Conceitos Demonstrados

- Criação de temas customizados
- Modo escuro (Dark Mode)
- Alternância entre temas
- ColorScheme
- Persistência de preferência de tema

## Funcionalidades

- Alternância entre tema claro e escuro
- Temas customizados
- Persistência da preferência

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como criar temas customizados
- Como implementar modo escuro
- Como alternar entre temas
- Como usar ColorScheme
- Boas práticas de design de temas
