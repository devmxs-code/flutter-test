# My Application

Aplicação Android nativa construída com Kotlin, apresentando drawer de navegação e múltiplos fragments.

## Funcionalidades

- Drawer de navegação
- Fragment inicial
- Fragment de galeria
- Fragment de slideshow
- Componentes Material Design
- Arquitetura ViewModel

## Como Começar

1. Abra o projeto no Android Studio
2. Sincronize os arquivos Gradle
3. Execute o app em um emulador ou dispositivo

## Estrutura do Projeto

- `app/src/main/java/com/example/myapplication/` - Código principal da aplicação
  - `MainActivity.kt` - Activity principal
  - `ui/home/` - Fragment inicial e ViewModel
  - `ui/gallery/` - Fragment de galeria e ViewModel
  - `ui/slideshow/` - Fragment de slideshow e ViewModel
- `app/src/main/res/` - Recursos (layouts, menus, valores)

## Tecnologias Utilizadas

- Kotlin
- Android SDK
- Material Design
- Navigation Component
- ViewModel
