# Aula 5 - Containers

Projeto de aprendizado Flutter demonstrando widgets Container e conceitos de layout.

## Funcionalidades

- Uso de widget Container
- Exemplos de layout e estilização
- Integração de assets de imagem

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Arquivo principal da aplicação com exemplos de containers

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
