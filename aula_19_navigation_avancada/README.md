# Aula 19 - Navigation Avançada

Projeto de aprendizado demonstrando navegação avançada, passagem de parâmetros e navegação nomeada no Flutter.

## Conceitos Demonstrados

- Navegação nomeada
- Passagem de parâmetros entre telas
- Navigator com rotas
- Navegação com retorno de dados
- Deep linking

## Funcionalidades

- Navegação entre múltiplas telas
- Passagem de dados
- Retorno de dados
- Rotas nomeadas

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como usar navegação nomeada
- Como passar parâmetros entre telas
- Como retornar dados de uma tela
- Como configurar rotas
- Boas práticas de navegação
