# Aula 12 - Forms e Validação

Projeto de aprendizado demonstrando criação de formulários e validação de dados no Flutter.

## Conceitos Demonstrados

- Criação de formulários com Form e GlobalKey
- TextFormField para campos de entrada
- Validação de campos com validator
- Controllers para gerenciar texto
- Feedback visual de erros

## Funcionalidades

- Formulário com campos de nome, email e senha
- Validação em tempo real
- Mensagens de erro personalizadas
- SnackBar para feedback de sucesso

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Formulário completo com validação

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como criar formulários no Flutter
- Como validar campos de entrada
- Como usar TextEditingController
- Como exibir mensagens de erro
- Boas práticas de UX em formulários
