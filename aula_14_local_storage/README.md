# Aula 14 - Local Storage

Projeto de aprendizado demonstrando armazenamento local de dados com SharedPreferences no Flutter.

## Conceitos Demonstrados

- Armazenamento local com SharedPreferences
- Salvar e recuperar dados
- Persistência de dados entre sessões
- Operações assíncronas com Future

## Funcionalidades

- Salvar texto localmente
- Carregar texto salvo ao iniciar o app
- Limpar dados salvos
- Feedback visual com SnackBar

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Exemplo de uso do SharedPreferences

## Tecnologias Utilizadas

- Flutter SDK
- SharedPreferences (armazenamento local)
- Material Design

## Aprendizados

- Como salvar dados localmente
- Como recuperar dados salvos
- Como limpar dados
- Persistência de dados entre sessões
- Boas práticas de armazenamento local
