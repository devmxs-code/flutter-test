# Aula 17 - Responsive Design

Projeto de aprendizado demonstrando design responsivo e adaptação a diferentes tamanhos de tela no Flutter.

## Conceitos Demonstrados

- Layout responsivo
- MediaQuery para detectar tamanho da tela
- LayoutBuilder para layouts adaptativos
- Breakpoints
- Adaptação para diferentes dispositivos

## Funcionalidades

- Layout que se adapta ao tamanho da tela
- Diferentes layouts para mobile e tablet
- Uso de MediaQuery e LayoutBuilder

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- Material Design

## Aprendizados

- Como criar layouts responsivos
- Como usar MediaQuery
- Como usar LayoutBuilder
- Como adaptar UI para diferentes tamanhos
- Boas práticas de design responsivo
