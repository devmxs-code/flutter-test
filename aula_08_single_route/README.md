# Aula 8 - Rota Única

Projeto de aprendizado Flutter demonstrando navegação com rota única.

## Funcionalidades

- Exemplo de navegação com rota única
- Padrões básicos de navegação
- Gerenciamento de rotas

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Aplicação com exemplo de rota única

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
