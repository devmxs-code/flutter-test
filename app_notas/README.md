# App Notas

Aplicação Flutter de notas com suporte a visualização web e capacidades de compartilhamento.

## Funcionalidades

- Criar e gerenciar notas
- Integração de visualização web
- Funcionalidade de compartilhamento
- Lançador de URL
- Shared preferences para persistência de dados
- Suporte a recebimento de intents de compartilhamento

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- Funcionalidade de visualização web e compartilhamento integrada

## Tecnologias Utilizadas

- Flutter SDK
- Flutter InAppWebView
- URL Launcher
- Share Plus
- Shared Preferences
- Receive Sharing Intent
