# AppBar Animado

Aplicação Flutter demonstrando um AppBar animado com funcionalidade de busca usando widgets Sliver.

## Funcionalidades

- AppBar customizado animado que colapsa e expande
- Integração de barra de busca
- Lista rolável baseada em Sliver
- Animações e transições suaves

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/search_page.dart` - Página principal de busca com conteúdo rolável
- `lib/silver_search_app_bar.dart` - Implementação do AppBar animado customizado
- `lib/search_bar.dart` - Widget de barra de busca
- `lib/background_wave.dart` - Animação de onda de fundo

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Widgets Sliver para comportamento de rolagem customizado
