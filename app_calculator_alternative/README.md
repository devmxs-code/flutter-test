# Calculadora Alternativa

Implementação alternativa de calculadora com suporte a temas e operações matemáticas.

## Funcionalidades

- Operações matemáticas básicas e avançadas
- Alternância de tema (claro/escuro)
- Design de UI moderno
- Avaliação de expressões

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/screens/calculator.dart` - Tela da calculadora
- `lib/widgets/` - Componentes de UI
- `lib/services/` - Lógica de negócio

## Tecnologias Utilizadas

- Flutter SDK
- Provider (Gerenciamento de Estado)
- Math Expressions
- Theme Provider
