# Clone WhatsApp

Aplicação Flutter estilo WhatsApp de mensagens com múltiplas telas.

## Funcionalidades

- Interface de tela de chat
- Tela de status
- Tela de chamadas
- Design de UI estilo WhatsApp
- Navegação entre telas

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação com roteamento
- `lib/chatscreen.dart` - Interface de chat
- `lib/status.dart` - Tela de status
- `lib/callscreen.dart` - Tela de chamadas
- `lib/chatroom.dart` - Implementação de sala de chat

## Rotas

- Tela de Chat (padrão)
- Tela de Status
- Tela de Chamadas

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Rotas nomeadas
