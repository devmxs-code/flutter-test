# Aula 20 - Gerenciamento de Estado com BLoC

Projeto de aprendizado demonstrando gerenciamento de estado usando o padrão BLoC (Business Logic Component) no Flutter.

## Conceitos Demonstrados

- Padrão BLoC
- Eventos e Estados
- BlocProvider e BlocBuilder
- Separação de lógica de negócio
- Testabilidade

## Funcionalidades

- Gerenciamento de estado com BLoC
- Eventos e estados reativos
- Arquitetura escalável

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## Tecnologias Utilizadas

- Flutter SDK
- flutter_bloc (padrão BLoC)
- equatable (comparação de objetos)

## Aprendizados

- Como implementar o padrão BLoC
- Como criar eventos e estados
- Como usar BlocProvider
- Como usar BlocBuilder
- Arquitetura escalável e testável
