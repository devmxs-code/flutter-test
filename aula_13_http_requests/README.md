# Aula 13 - HTTP Requests

Projeto de aprendizado demonstrando requisições HTTP e consumo de APIs REST no Flutter.

## Conceitos Demonstrados

- Requisições HTTP GET
- Tratamento de respostas JSON
- Gerenciamento de estado assíncrono
- Tratamento de erros
- Loading states

## Funcionalidades

- Busca de dados de uma API pública
- Exibição de dados JSON
- Indicador de carregamento
- Tratamento de erros

## Como Usar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute o app:
   ```bash
   flutter run
   ```

## API Utilizada

Este projeto utiliza a API pública JSONPlaceholder:
- Endpoint: `https://jsonplaceholder.typicode.com/posts/1`

## Estrutura do Projeto

- `lib/main.dart` - Exemplo de requisição HTTP

## Tecnologias Utilizadas

- Flutter SDK
- HTTP package (requisições HTTP)
- Material Design

## Aprendizados

- Como fazer requisições HTTP no Flutter
- Como trabalhar com dados JSON
- Como gerenciar estados assíncronos
- Como tratar erros em requisições
- Como exibir indicadores de carregamento
