# Aula 7 - Iniciando Rotas

Projeto de aprendizado Flutter introduzindo conceitos de navegação e roteamento.

## Funcionalidades

- Implementação básica de roteamento
- Navegação entre telas
- Configuração de rotas

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação com configuração de rotas

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
- Rotas nomeadas
