# App Hábitos

Aplicação Flutter para rastreamento de hábitos diários e manutenção de consistência.

## Funcionalidades

- Rastrear múltiplos hábitos
- Tiles visuais de hábitos
- Interface simples e intuitiva
- Mensagens motivacionais

## Como Começar

1. Instale as dependências:
   ```bash
   flutter pub get
   ```

2. Execute a aplicação:
   ```bash
   flutter run
   ```

## Estrutura do Projeto

- `lib/main.dart` - Ponto de entrada da aplicação
- `lib/home_page.dart` - Página principal exibindo tiles de hábitos
- `lib/util/habit_tile.dart` - Widget de tile de hábito reutilizável

## Tecnologias Utilizadas

- Flutter SDK
- Material Design
